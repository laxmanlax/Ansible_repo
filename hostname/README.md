Role Name
=========
This role helps to set hostname in all servers , here you just have to define what hostname you want with IP

EXAMPLE :

                  host:
                    web1:  
                      ip: 10.0.15.21  
                    web2:
                      ip: 10.0.15.22
                    web3:
                      ip: 10.0.15.23
                    web4:
                      ip: 10.0.15.24
